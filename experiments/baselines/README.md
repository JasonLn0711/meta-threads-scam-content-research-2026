# Baseline Experiments

Use this directory for keyword, rule, OCR-augmented, classifier, LLM-assisted, and hybrid triage experiment logs.

Start with simple text rules before adding heavier methods.

## Current Logs

| Experiment | Purpose |
|---|---|
| `0001-rule-baseline-v1.md` | Protocol for the first rule baseline on the authorized pilot. |
| `0002-synthetic-rule-baseline-dry-run.md` | Synthetic-only dry run showing the rule variants and first calibration adjustment. |
| `0003-calibration-workbench.md` | Workbench protocol for threshold profiles, evidence variants, and reviewer worksheet export. |
| `baseline_experiment_plan.md` | Modular rule-baseline comparison plan for text, replies, OCR, all signals, and threshold variants. |

Generated local prediction artifacts belong under `experiments/baselines/outputs/`, which is ignored by git.
